🛒 쿠팡 오픈 API 자동화 툴

브라우저에서 간편하게 테스트하세요

📌 사용 전 확인사항:
1. 애플리케이션이 실행 중이어야 합니다 (포트 8080)
2. application-secret.properties에 쿠팡 API 설정이 되어 있어야 합니다:
    - coupang.api.vendorId (업체코드)
    - coupang.api.accessKey
    - coupang.api.secretKey
3. 쿠팡은 HMAC 서명 기반 인증을 사용하므로 별도의 토큰 발급이 필요 없습니다.

1애플리케이션 상태 확인

애플리케이션이 정상적으로 실행 중인지 확인합니다.

2엑셀 파일로 상품 일괄 업로드

엑셀 파일(.xlsx)을 업로드하여 쿠팡에 상품을 자동으로 등록합니다.

📋 엑셀 파일 형식:
필수 컬럼: 상품명, 카테고리, 판매가, 재고수량, 상세설명, 대표이미지URL
선택 컬럼: 추가이미지URL1, 옵션명1, 옵션값1, 옵션명2, 옵션값2, 브랜드, 모델명, 바코드, 배송비, 배송방법, 원산지, 제조사, 과세구분, 판매상태, 전시상태

⚠️ 주의사항:
- 카테고리는 쿠팡 카테고리 코드(displayCategoryCode)를 입력해야 합니다
- 반품 정보는 기본값이 사용되며, 실제 운영 시에는 설정이 필요할 수 있습니다
.xlsx 또는 .xls 파일만 지원
