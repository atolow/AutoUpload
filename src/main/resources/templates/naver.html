🛍️ 네이버 스마트스토어 자동화 툴

브라우저에서 간편하게 테스트하세요

📌 사용 전 확인사항:
1. 애플리케이션이 실행 중이어야 합니다 (포트 8080)
2. application.properties에 Client ID/Secret이 설정되어 있어야 합니다
3. 판매자 ID 확인 방법:
    → 네이버 스마트스토어 판매자센터 로그인
    → 스토어 설정 → 스토어 정보 → 판매자 ID 확인

1애플리케이션 상태 확인

애플리케이션이 정상적으로 실행 중인지 확인합니다.

2스토어 등록 / 토큰 갱신

✨ 자동 모드: 스토어가 이미 등록되어 있으면 저장된 정보로 자동 토큰 갱신합니다.
📝 최초 설정: 스토어가 없으면 최초 1회만 판매자 ID를 입력하세요.

ℹ️ 현재 등록된 스토어가 있습니다.
버튼을 클릭하면 저장된 정보로 자동 토큰 갱신이 진행됩니다.
기본값: SELF
📌 판매자 ID 확인 방법:
1. 네이버 스마트스토어 판매자센터 로그인
2. 스토어 설정 → 스토어 정보 메뉴 클릭
3. 판매자 ID 또는 스토어 ID 확인

⚠️ 주의: Client ID (예: 7gz2KKBlKrNUFaAOt1urVB)가 아닌
            실제 판매자 ID를 입력하세요!

3등록된 스토어 확인

현재 등록된 스토어 정보를 확인합니다.

4엑셀 파일로 상품 일괄 업로드

엑셀 파일(.xlsx)을 업로드하여 네이버 스토어에 상품을 자동으로 등록합니다.

📋 엑셀 파일 형식:
필수 컬럼: 상품명, 카테고리, 판매가, 재고수량, 상세설명, 대표이미지URL
선택 컬럼: 추가이미지URL1, 배송비, 배송방법, 브랜드, 모델명, 제조사, 원산지, 과세구분, 판매상태, 전시상태
회사별 엑셀 컬럼 구조를 구분하기 위한 회사명 (기본값: CompanyTest)
.xlsx 또는 .xls 파일만 지원
